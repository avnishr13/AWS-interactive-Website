Amazon Web Services (AWS)

AWS is the world's most comprehensive and broadly adopted cloud platform, offering over 200 fully featured services from data centers globally.

  • ✓ EC2 (Elastic Compute Cloud): Scalable virtual servers.
  • ✓ S3 (Simple Storage Service): Object storage built for retrieval of any amount of data.
  • ✓ Lambda: Run code without provisioning or managing servers.
  • ✓ RDS: Managed relational database service.

DevOps Methodology

DevOps is the combination of cultural philosophies, practices, and tools that increases an organization’s ability to deliver applications and services at high velocity.

  • ➜ Automation: Reducing manual toil in testing and deployment.
  • ➜ Containerization: Using Docker & Kubernetes for consistent environments.
  • ➜ Microservices: Building applications as a suite of small services.
  • ➜ Security (DevSecOps): Integrating security into the development workflow.
